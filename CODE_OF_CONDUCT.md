# Code of Conduct

This project adopts the [Contributor Covenant v3.0](https://www.contributor-covenant.org/version/3/0/code_of_conduct/) as its Code of Conduct.

Please read the full text at the link above. By participating in this project (issues, pull requests, discussions), you agree to abide by its terms.

## Enforcement

Reports of unacceptable behavior may be sent to the project maintainers at **arturo.rodas.gonzales@gmail.com**. All reports will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of reporters.

For details on Enforcement Guidelines, see the [official Contributor Covenant v3.0 Enforcement section](https://www.contributor-covenant.org/version/3/0/code_of_conduct/#enforcement).
